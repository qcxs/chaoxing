# 超星学习通修改教程

> 使用 MT 管理器修改超星学习通 6.7.2 版本的详细教程。涵盖脱壳修复、功能修改、JS 注入等完整流程。

## 引言

此仓库详细记录了使用 [MT 管理器](https://mt2.cn/) 修改超星 [学习通](https://app.chaoxing.com/) 6.7.2 版本的具体流程。在此之前，全网中**只**找到**可以拖动进度条**的 4.3.4 版本和少量相关教程，可见相关资源匮乏。

::: tip 提示
- 有些操作需要 MT 会员，可使用 [NP 管理器](http://normalplayer.top/) 代替
- 阅读此教程需要一定的逆向知识
- 遇到问题可以使用 AI 辅助分析
:::

## AI 辅助修改

由于 ProGuard 的混淆缘故，即使知道怎么修改，不同版本修改的位置也不一样。但借助 AI 可以快速定位问题，除了 [脱壳修复](/guide/unpack-repair) 需要手动操作外，其他的分析都可以交给 AI 来做。

### 推荐工具

- **[APK MCP](https://mt2.cn/guide/reverse/apk-mcp.html)** — 手机端使用
- **[jadx-ai-mcp](https://github.com/zinja-coder/jadx-ai-mcp/releases)** — 电脑端使用

### 通用提示词

```text
# 1、先让ai了解项目
调用jadx/mt mcp服务，打开apk，了解apk信息

# 2、限定ai给出的修改意见，目前mcp提供的修改功能都不好用，推荐自行修改
你无需修改，只负责提供意见。修改意见：反编译代码并不好修改，常用的修改方案有
更改某处判断（只需更改某行代码）、清空某个方法（阻止方法执行）、
修改方法返回值（只需在末尾添加一行代码，且只限基本类型），
像在中间添加调用方法等，也可以使用但比较麻烦，
因为涉及到smali的寄存器，所以优先级并不高。
```
